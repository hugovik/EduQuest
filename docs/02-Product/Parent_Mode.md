# Parent Mode

---

# Purpose

Parent Mode provides parents and guardians with meaningful insight into their child's educational journey while preserving the child's sense of ownership and independence.

Parent Mode is designed to support, encourage, and celebrate learning—not to monitor every action or create unnecessary pressure.

Parents become partners in the adventure.

---

# Vision

Parents should feel informed without needing to constantly supervise.

Children should feel trusted rather than monitored.

The Parent Dashboard should answer one simple question:

> "How is my child growing?"

instead of:

> "How many questions did they answer today?"

The focus is growth, not surveillance.

---

# Core Principles

Parent Mode should always be:

- supportive
- informative
- encouraging
- respectful
- privacy-conscious
- easy to understand

It should strengthen family learning rather than replace conversations.

---

# Parent Goals

Parents want to:

- understand progress
- celebrate achievements
- identify strengths
- recognize challenges
- encourage consistency
- participate in learning

EduQuest helps achieve these goals without overwhelming parents with unnecessary data.

---

# Dashboard Overview

The Parent Dashboard provides a high-level summary of the child's progress.

It includes:

- Current Level
- Tree Stage
- Total XP
- Weekly Activity
- Learning Streak
- Worlds Unlocked
- Certificates Earned
- Recent Achievements

Parents should understand their child's progress within a few seconds.

---

# Learning Overview

The dashboard summarizes learning by subject.

Example:

Reading

- Stories Completed
- Reading Time
- Comprehension Progress

Writing

- Writing Activities
- Vocabulary Growth
- Creative Exercises

Mathematics

- Challenges Completed
- Accuracy
- Confidence Level

Science

- Experiments
- Discovery Activities

This allows parents to quickly identify balanced development.

---

# Weekly Summary

Every week EduQuest generates a summary.

Example:

```
This Week

12 Quests Completed

4 Reading Adventures

3 Math Challenges

2 Writing Activities

1 Science Experiment

420 XP Earned

Tree Grew to Young Tree

New Achievement:
Story Keeper
```

Summaries focus on accomplishments rather than deficiencies.

---

# Milestones

Parents receive notifications for significant milestones.

Examples:

- New Tree Stage
- World Unlocked
- New Certificate
- Companion Evolution
- Reading Streak
- Birthday Celebration

Notifications should be infrequent and meaningful.

---

# Recommendations

EduQuest provides gentle recommendations.

Examples:

- Reading has been excellent this week.
- Consider a few more writing adventures.
- Science Laboratory has recently unlocked.
- Spark discovered a new quest.

Recommendations should encourage rather than criticize.

---

# Learning Trends

Parents can review long-term progress.

Examples include:

- Reading growth
- Mathematics confidence
- Learning consistency
- Weekly activity
- Tree progression
- XP growth

Visual graphs should remain simple and child-friendly.

---

# Family Goals

Parents may create optional goals.

Examples:

- Read together twice this week.
- Complete five adventures.
- Explore a new world.
- Finish one story.

Goals should encourage shared experiences rather than competition.

---

# Family Challenges

Future versions may include family activities.

Examples:

- Weekend Reading Adventure
- Family Puzzle Night
- Nature Exploration Challenge
- Story Creation Evening

Parents become active participants in learning.

---

# Screen Time

Parents may configure healthy learning limits.

Examples:

- Daily learning goal
- Suggested break reminders
- Bedtime pause
- Weekend flexibility

EduQuest encourages healthy habits rather than maximizing screen time.

---

# Privacy Controls

Parents control:

- account management
- connected devices
- teacher access
- data sharing
- notifications

Children should still feel ownership over their personal world.

---

# Reports

Parents may download reports including:

- Weekly Summary
- Monthly Progress
- Learning Portfolio
- Certificates
- Achievement History

Reports should be suitable for school meetings if needed.

---

# Multiple Children

Parent Mode supports multiple child profiles.

Dashboard includes:

```
Emma

Young Tree

Level 8

------------------

Noah

Sapling

Level 5

------------------

Lena

Growing Tree

Level 12
```

Switching between children should be immediate.

---

# Parent Notifications

Meaningful notifications include:

- New Certificate
- Tree Evolution
- Story Completed
- Major Achievement
- Weekly Summary Available

Parents should never receive excessive notifications.

---

# Conversation Starters

EduQuest helps parents continue learning offline.

Examples:

"Ask your child what Spark discovered today."

"Can they tell you about the story they read?"

"What was the hardest puzzle today?"

These prompts strengthen family engagement.

---

# Accessibility

Parent Mode should support:

- mobile phones
- tablets
- desktop browsers
- dark mode
- readable typography
- accessible charts

Parents should be able to review progress in less than five minutes.

---

# Technical Features

Parent Mode includes:

- Secure Authentication
- Multiple Child Profiles
- Progress Analytics
- Notification Preferences
- PDF Report Generation
- Certificate Downloads
- Account Settings

Future versions may integrate with school information systems.

---

# Future Expansion

Potential future additions include:

- AI-generated progress summaries
- Personalized learning recommendations
- Family learning calendar
- Shared reading lists
- Goal planning
- Teacher messaging
- Parent discussion groups

These features should always support—not replace—family involvement.

---

# Success Criteria

Parent Mode succeeds when parents:

- understand their child's progress
- celebrate achievements together
- feel confident about educational development
- remain engaged without feeling overwhelmed

Children should feel supported rather than monitored.

---

# Vision Statement

Parent Mode exists to strengthen the partnership between children, families, and learning.

It transforms educational progress into meaningful family conversations, shared celebrations, and lasting memories.

The greatest achievement is not simply helping a child learn.

It is helping families enjoy learning together.
