# EduQuest Core v0.1.0

First working vertical-slice starter for **EduQuest Adventures**.

## Includes

- React + Vite frontend
- FastAPI backend
- Seeded child profile
- Tree House home screen
- World Map
- Reading Forest quest
- XP reward flow
- Certificate placeholder
- Data-driven starter structure

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
