# User Journeys

---

# Purpose

This document defines the major user journeys within EduQuest.

A user journey describes the complete experience of a player, parent, or teacher while interacting with the platform.

Understanding these journeys ensures that individual features work together to create a cohesive product experience.

---

# Overview

EduQuest supports three primary user journeys:

- Child Journey
- Parent Journey
- Teacher Journey

The child journey is the primary focus of the platform.

---

# Child Journey

## Goal

Transform daily learning into an engaging adventure.

---

## Journey 1 — First Launch

### Starting Point

The child opens EduQuest for the first time.

### Experience

- Welcome animation
- Introduction to Professor Owl
- Introduction to Spark Dragon
- Create player profile
- Choose avatar
- Receive first Tree Seed
- Enter the Tree House

### Outcome

The player feels ownership of their world.

---

# Journey 2 — Daily Login

### Starting Point

The player opens EduQuest.

### Experience

- Welcome back message
- Tree House loads
- Tree of Growth reflects current progress
- Companion greets the player
- Daily quest becomes available

### Outcome

The player immediately knows what to do next.

---

# Journey 3 — Begin Adventure

### Starting Point

The player selects today's quest.

### Experience

- Professor Owl introduces the story
- Educational objective is explained
- Player enters the learning world

### Outcome

Learning begins naturally through gameplay.

---

# Journey 4 — Complete Learning Activity

### Experience

The player:

- reads
- writes
- solves problems
- answers questions
- explores

The activity adapts to the player's ability.

---

### Completion

When the quest finishes:

- XP increases
- Tree grows
- Companion reacts
- Rewards appear
- Achievements unlock (if applicable)

### Outcome

The player experiences immediate positive feedback.

---

# Journey 5 — Return Home

The player returns to the Tree House.

The Tree House now reflects the completed adventure.

Possible changes:

- larger tree
- new decoration
- unlocked achievement
- companion dialogue
- new story event

The world remembers what happened.

---

# Journey 6 — Long-Term Progress

Over days and weeks the player:

- levels up
- unlocks worlds
- grows companions
- restores forgotten lands
- discovers secrets

Learning creates a persistent sense of progress.

---

# Parent Journey

## Goal

Understand and encourage the child's progress.

---

### Parent Login

Parent signs in to Parent Mode.

Dashboard displays:

- weekly activity
- completed quests
- learning streak
- strengths
- recommendations

---

### Celebrate Success

Parents receive milestone notifications.

Examples:

- New Tree Stage
- Reading Milestone
- Seven-Day Streak
- New Achievement

The application encourages shared celebration.

---

### Support Learning

Parents may:

- set goals
- choose learning focus
- approve screen time
- review reports

Parents guide rather than control.

---

# Teacher Journey

## Goal

Support classroom learning.

---

### Teacher Dashboard

Teachers can:

- create classroom
- invite students
- assign quests
- review completion
- monitor progress

---

### Classroom Review

Teachers see:

- participation
- learning trends
- struggling students
- achievements
- completed assignments

---

### Encourage Students

Teachers celebrate:

- improvement
- persistence
- consistency
- collaboration

The platform supports positive classroom culture.

# Journey Principles

Every user journey should:

- begin with clear purpose
- minimize confusion
- reward exploration
- provide immediate feedback
- encourage continued learning
- leave the user wanting to return

---

# Future Journeys

Future releases may include:

- Family Challenges
- Classroom Competitions
- Seasonal Events
- Community Projects
- AI Tutoring Sessions
- Companion Storylines
- World Expeditions

Each future journey must remain consistent with the core philosophy of learning through adventure.

---

# Definition of Success

A successful user journey is one where the player finishes a session feeling:

- accomplished
- curious
- confident
- excited to return

Learning should always conclude with anticipation for the next adventure.
